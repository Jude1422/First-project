What is HTML

HTML stands for hyperText Markup language.It's a markup language that web developers use to stucture and describe the content of a webpage(not a programming language) HTML consists of elements that describe different types of content:paragraphs,links,heading,images,video,etc. Web browsers understand HTML and render HTML code as websites

in HTML,each element is made up of 3 parts

  1. The opening tag
  2. The closing tag
  3. The actual element
